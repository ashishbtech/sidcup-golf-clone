# Sidcup Family Golf - UI/UX Clone ⛳

A frontend UI/UX clone of the Awwwards-winning [Sidcup Family Golf](https://sidcupfamilygolf.com/) website. The goal of this project was to practice advanced layout structures and recreate their highly interactive design, custom cursors, and smooth scroll animations from scratch.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Styling, responsive design, and layout structuring.
* **Vanilla JavaScript:** DOM manipulation, custom cursor logic, and event handling.
* **GSAP & ScrollTrigger:** Used for complex, scroll-based animations, text reveals, and element scaling.

## ✨ Key Features
* **Custom Interactive Cursor:** A custom cursor element that follows the mouse and scales up/changes color when hovering over the navigation bar.
* **Scroll Animations:** Complex, scroll-triggered animations including navigation bar shrinking, image scaling, and parallax effects.
* **Video Background:** Seamless integration of an auto-playing background video to match the original site's dynamic hero section.
* **Hover Effects:** Interactive 3D tilt effects on the cards section.

## 🚀 Run Locally

To view this project on your local machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/ashishbtech/sidcup-golf-clone.git](https://github.com/ashishbtech/sidcup-golf-clone.git)
   ```

2. Navigate to the project directory:
   ```bash
   cd sidcup-golf-clone
   ```

3. Open `index.html` in your web browser, or use VS Code's "Live Server" extension to run it on a local development server.

## 🎯 What I Learned
* Handling complex, scroll-based animations and timeline scrubbing using GSAP.
* Creating and managing custom mouse followers using JavaScript math (`clientX`/`clientY`).
* Structuring clean, maintainable CSS for an animation-heavy webpage.
